JLPT N1擬声語と擬態語

✨擬声語&擬態語✨

擬声語

聞こえる物音を表す言葉

擬態語

聞こえないものや身振りなどの状態を表す言葉

✨見る✨

きょろきょろ

じっと

じろじろ

ちらちら

単語 例文 説明
きょろきょろ 東京に初めて来たとき、きょろきょろしていたら変な人だと思われた。 いろんなところを見る様子。
じっと 話した事もないのにじっと見てくる男の子がいます。私のことが好きなのかしら。 ずっと一つのものを見ていること。
じろじろ 何さっきからじろじろ見ているの？本当に失礼ね。 一つのもののいろんなところを見る。
ちらちら おい！ちらちらどこを見ているんだ！まだ授業は終わってないぞ。 時々違う場所を見る。

✨感じる✨

さらさら

つるつる

ふわふわ

すべすべ

単語 例文 説明
さらさら 美容室帰りのサラサラ髪。 スムーズに指が髪の毛に通る感じ。
つるつる 温泉に入ると肌がツルツルになりますよね！ 滑りやすいこと。
ふわふわ 干したばかりの布団はふわふわしていてとて気持ちいい。 空気をいっぱい含むんで、やわらかい。
すべすべ 赤ちゃんの肌ってほんとすべすべだよね。 肌が良い状態のこと。

✨感じる ２回目✨

ざらざら

かさかさ

べたべた

ねばねば

ぬるぬる

単語 例文 説明
ざらざら もう夏になるし、かかとのザラザラをなくしたい。 硬いしなんかごそごそするし。
かさかさ 肌が乾燥する季節突入。毎年この時期になると肌がカサカサになる。 水分不足が原因の肌荒れ。落ち葉。
べたべた 汗でベタベタだから早くシャワーを浴びたい。 気持ち悪いくらいの粘着感。
カップルとか人がいちゃついている時。
ねばねば とろろ丼とか納豆は、ねばねばしているけど健康にいいしとてもおいしい。 食べ物に使うことが多いです。
ぬるぬる なめこはなぜヌルヌルしているんですか？ ちょっと光っていて、光沢があって、つやつやしていて、ぬめぬめ、ねばねばするもの。

✨食べる✨

こってり

あっさり

さっぱり

ひりひり

ツーン

単語 例文 説明
こってり こってりしたラーメンは大好きだけど、次の日が大変だからね。 油が多くて、味も濃い、どろどろ。
あっさり でもラーメンは食べたいから、今日はあっさりしたラーメンにしとこう。 「こってり」の正反対。
さっぱり 夏場でもレモンをかけることでさっぱりといただけます。 すっきりしていて重たくなくてちょっと酸味のあるもの。冷たいものが多い。
ひりひり・ぴりぴり 辛い物を食べたら舌がひりひりしてしょうがない。水が欲しい！ 熱い状態。表面が痛い、浅い痛み。やけど、日焼けの時も「ひりひり」という。
ツーン わさびが入ったお寿司を食べたら、鼻がツーンと痛くなった。 わさび。

✨食べる　２回目✨

しこしこ

もちもち

しゃきしゃき

ほかほか

かりかり

単語 例文 説明
しこしこ この麵はしこしこしていてとても美味しかったです。 弾力がある。嚙み応えがある。太くていっぱい嚙ないといけない。こしがある。
もちもち この大福すごくもちもちだよ。 弾力がある。
しゃきしゃき このサラダ、レタスがとてもしゃきしゃきしていておいしいです。 フレッシュで水分も多い。野菜に使われることが多い。
ほかほか 炊き立てのほかほかご飯で幸せいっぱい。 あったかくて、やわらかくて、
かりかり ベーコンはカリカリに焼いたほうがおいしい。 しっかりと長い時間焼いて、水分が抜けた状態。かたい状態。

✨食べる　３回目✨

がつがつ

ぱくぱく

もぐもぐ

もりもり

単語 例文 説明
がつがつ フレンチレストランでがつがつ食べるのはやめてください。 すごい勢いで食べている。欲張りなようす。
ぱくぱく 野菜嫌いのケントも、カレーに入れるとぱくぱく食べる。 元気に、嫌がらずに食べている。
もぐもぐ 口をもぐもぐさせてる。 口に物を入れて動かしている（嚙んでいる）状態。口の中にたくさん入れているような感じ！
もりもり モリモリ食べて暑さを吹き飛ばしましょう！ 元気よくたくさん食べている状態。

✨飲む✨

ぐっと

ごくごく

ちびちび

がぶがぶ

単語 例文 説明
ぐっと 皆：はい！一気！一気！一気！
部長：君もぐっと飲んで。ぐっとぐっと！		 勢いがあって一気に。
ごくごく ごくごくとおいしそうに飲むね。私も飲みたくなっちゃったよ。 のどを鳴らしておいしそうに飲む様子。
ちびちび のりこ：けんとくん！ちびちび飲んでないで、男らしくぐっとね！
けんと：あ、お酒に弱いので。		 酒などを少しずつ飲む様子。少しずつ楽しみながら飲む。
がぶがぶ 試合後、シン選手は水をがぶがぶと飲み始めた。 水を大量に飲む感じ（カバのように）。
口を大きく開けてたくさん入れる。

✨言う✨

おずおず

がんがん

ぶつぶつ

べらべら

ぼそぼそ

くどくど

単語 例文 説明
おずおず 「あの、すみません。私がこのパソコン壊しました。」と彼はおずおず言った。 言い難そうな様子。申し訳なさそうな感じ。
がんがん そんなにガンガン言われたら、やる気もなくなっちゃうよ。 ずっと続けて、何度も強く言う。
ぶつぶつ 彼はいつも仕事中ぶつぶつと独り言を言っている。 周りに聞こえないような感じ。
文句や不満を言う。
べらべら 秘密をベラベラしゃべる人は、信用できません。 何でもいっぱいしゃべる人。
ぼそぼそ 下を向いてぼそぼそ話していると、何が言いたいのか、わかりません。 音が小さいだけ。
くどくど くどくどと何回もしつこいわよ！ 何回もしつこく言うこと。

✨寝る✨

うとうと

ぐうぐう

すやすや

ぐっすり

こっくりこっくり

単語 例文 説明
うとうと 授業中にうとうとしていると体がビクッとなってしまった。 ちょっとの間浅く寝るさま。浅い。
ぐうぐう 猫ってぐうぐう寝るの？あまり音出さない気がするけど。 寝ているときの音。
お腹が空いたとき。
すやすや 子供がすやすやと寝る姿を見ると育児の疲れも吹っ飛びます。 静かで穏やかな眠り。
ぐっすり 最近、寝ても疲れが取れないのです。ぐっすり寝る方法を教えてください。 深い。
こっくりこっくり 彼は夜の電車のつり革にぶら下がりながらこっくりこっくりしている。 頭が揺れながら寝ている状態。

✨動き１早い✨

さっさと

すらすら

てきぱき

はきはき

きびきび

単語 例文 説明
さっさと 明日は試験だよ！いつまでも寝てないでさっさと勉強しなさい！ 時間を無駄にせずに早く動く様子。
すらすら 日本語を勉強し、思ったことをすらすら言えるようになりたい。 話し方が流暢。ぺらぺら。
止まらず、スムーズにこなすこと。
てきぱき 仕事がテキパキ出来る人ってうらやましい。私もそうなりたいな。 効率的に仕事をする様子動作やものごと。
はきはき のりこさんはいつもはきはきしていますね。 はっきりして、元気な。
きびきび 店長に怒られないようにきびきび働く。 すごく元気でいきいきしていて、きもちがよいさま、動作や態度。

✨動き２遅い✨

のろのろ

ゆっくり

のんびり

だらだら

単語 例文 説明
のろのろ 最近駅でのろのろ歩きながらスマホをしている人が多くて、よく人ぶつかります。 遅い状態。動きが鈍い様子。
ゆっくり 慌てないで。落ち着いて、ゆっくり話して。 遅くてゆとり、余裕があるさま。
のんびり 今日は日曜日だし、ベランダ日差しでも浴びながらのんびり、まったりと時間を過ごそうと～ 時間に余裕があってゆっくりすることができる。
だらだら だらだらした生活から抜け出し、充実した生活を送りたい。 適当にという感じ。生活リズムが乱れているような様子。

✨動き３歩く✨

ぶらぶら

よたよた

とぼとぼ

よちよち

すたすた

てくてく

単語 例文 説明
ぶらぶら 家の近くでぶらぶらしていたら、ケント君にばったりあった。 別の目的がなくのんびりと歩くさま。
ぶらぶら遊ぶ（悪い意味）
ふらふら→力が入らなかったりして体が揺れ動くさま。
よたよた 赤ちゃんがよたよたと歩いている。可愛い！ 足がふらふらして危ないさま。
お年寄り
疲れている人にも使う。
とぼとぼ 彼は好きな人に告白したが、きっぱり断わられ、とぼとぼと家まで歩いて行った。 元気がなく寂しそうに歩くさま。
よちよち 赤ちゃんがよちよち歩く姿はとてもかわいい。 小股で歩く様子。
すたすた 田舎の人はのろのろと歩くが、東京の人はすたすたと歩く。 わき目もふらず、早く歩くさま。
てくてく 学校終わりの小学生が帰り道をテクテク歩いている。 乗り物に乗らずひたすら歩くさま。
歩いている時以外には使わない。

✨人の動き４✨

うろうろ

こそこそ

こつこつ

せっせと

ぺこぺこ

単語 例文 説明
うろうろ 知らない男の人が家の前をうろうろしていて、なんだか気味が悪い。 歩き回る様子。怪しい、怖い。
目的は無い。意味も無く、理由も無く。
こそこそ 親に隠れて、コソコソとゲームをやっていたがばれてしまった。 人に知られないようにひそかに物事をするさま。
こそこそと話す。
こつこつ コツコツ練習したら、上手になるよ。 地道に、ちょっとずつ、毎日続ける。努力する様子。
せっせと 日曜日も休まず、せっせと働く。 休まずに一生懸命に事を行うさま。
ぺこぺこ 彼は出世のため、いつも上司にも部下にもぺこぺこしている。 頭を何度も下げる様子。
お腹ぺこぺこ。

✨人の動き５落ち着かない✨

あたふた

ばたばた

そわそわ

そそくさ

ぐずぐず

単語 例文 説明
あたふた 「5分後にお客さん来るよ。」と聞いた紀子先生は、あたふたしながら掃除を始めた。 慌てているが急いで行動しているさま。
ばたばた やっと仕事が終わった。今日は朝からずっとばたばたしていた。 歩き回りながら忙しくて慌てている様子。
そわそわ 授業の終わりが近づくと、学生たちは授業に集中せず、そわそわし始めた。 落ち着かない様子。
他のことが気になって、心が落ち着かないさま。
ドキドキ似ている。
そそくさ 友達がいない彼は、大学の授業が終わったとたんそそくさと帰った。 すぐに。硬い表現。
ぐずぐず 母：ぐずぐずしているとおいていくよ！ 動きがのろくてあまり物事が進まないさま。
子供に使うことが多い。

✨人の感情１驚き✨

どきっとする

はっとする

どきどきする

ぽかんとする

ぎょっとする

単語 例文 説明
どきっとする 男性は髪の長い女性にドキッとすることが多い。 興奮しらり、驚いたりしていきなり心臓を打つさま。
どっきり
はっとする バスでうとうとしていたけど、アナウンスではっと目覚めたらちょうど目的地だった。 何かを急に気づいて驚くさま。
「ハッ」は息の音
どきどきする 何回も練習したけど、人前に立って話し始めたらどきどきして大変だった。 緊張、運動、不安で、心臓が早く強く打つ様子。
ぽかんとする いつも口をぽかんと開けていたら、だらしないよ。 驚いて口が開いている様子。
ぎょっとする 鏡を見ながら髪を直していたら、後ろに紀子先生がいてぎょっとした。 考えてもなかったことがいきなり起こって驚いたり怖いがる様。

✨人の感情２怖い✨

ぞっとする

びくびくする

ひやひやする

おどおどする

ぶるぶるする

単語 例文 説明
ぞっとする 私は高いところが怖いので、12階から見下ろすとぞっとする。 寒さや恐怖で背筋が寒くなるさま。
「かんがえだけで」、「想像しただけで」と一緒に使うことが多い。
びくびくする ボールを投げたら父の車に当たってひびが入ってしまった。
父が気付くまでびくびくしながら部屋に隠れていた。		 悪いことが起こると思って、常に怖がっている様子。
体を震える。
ひやひやする 落ちてしまいそうで、背中が汗でびっしょりするぐらいひやひやした。 怖い、冷たい、寒い。
悪いことが起こると思って不安がる様子。
おどおどする いつもおどおどしていないで、はきはき話しましょう。 怖くて自信を無くし、落ち着かないさま。
ぶるぶるする 紀子先生の顔を見た子犬が、ぶるぶる震えだした。 寒さや恐怖などのために震えるさま。

むかむか

かっ

むっ

ぷんぷん

単語 例文 説明
いらいら 20分待っても、注文した料理が出てこないので、イライラしてきた。 思いどおりにいかなくて、神経が高ぶること。
いらっとする。
むかむか 今朝、親に怒られてむかむかしている。 後になってからも腹が立つ気持ち。
心の奥底から怒りがこみ上げてくるさま。
吐き気がして気持ちの悪いさま。
かっ 彼女は短気ですぐかっとなってしまう。 急に怒り出す。
キレる。
むっ 私のミスで彼女はムッとしているけど、どこが悪かったのかを教えてほしい。 心の中で怒るさま。
ぷんぷん もぅ。プンプンだよ。プンプン。 口を膨らませながら怒っているような。
女の子が言う。
激おこぷんぷん丸。

✨人の感情４いい気持ち✨

いきいき

うきうき

わくわく

すっきり

ほっと

単語 例文 説明
いきいき 彼はアニメの話になると、とてもイキイキした顔になる。 明るくて元気なさま。
うきうき 春休みの旅行のことを考えただけウキウキする。 足が地面から離れてしまいそうなくらい楽しい感覚。
わくわく ワクワクしながら宝くじの当選結果を確認した。 これから起こることに対し期待するさま。
すっきり ずっと胸にためていたことを全部言葉に出せてすっきりした。 余計なものがくなり、気持ちの良いさま。
ぐっすり寝た後とかしっかり掃除した後もよく使う。
ほっと 迷子になったと思っていた息子が無事に帰ってきてほっとした。 よかった、胸をなでおろす感じ。
安心する。

✨人の感情５愛✨

めろめろ

いちゃいちゃ

でれでれ

ベタベタ

キュン

単語 例文 説明
めろめろ 私はもう彼にメロメロなの。毎日会っても全然飽きないぐらい。 愛に酔って理性を失ったさま。
いちゃいちゃ 高校生が朝から公園でいちゃいちゃしているのを見て嫌な気持ちになった。 カップル同士が人前でスキンシップをしているさま。
いちゃつく。
でれでれ けんと先生は好きな女性の前でデレデレになるらしい。 彼女、彼氏の前でだらしなくなるさま。
甘くなる。何でも許す。
ベタベタ いくら新婚でも、公共の場でベタベタ触りあうのはだめだ。 主にカップルがくっつきあうさま。
ベタつく。
キュン あなたに「愛してる」と言われるたびにキュンとするの。 ときめく感情。
胸キュン。
キュンキュン

✨空間１余裕がない✨

ぎっしり

ぎゅうぎゅう

ぴちぴち

ぴったり

単語 例文 説明
ぎっしり 予定がぎっしり詰まっていまして、空いているのは月曜の午前だけです。 すきまなく詰まっているさま。
予定、荷物、知識。
ぎゅうぎゅう 朝の地下鉄はぎゅうぎゅうでもう息も出来ないぐらいだ。 無理に詰め込むさま。
荷物、建物、乗り物。
ぴちぴち 太り過ぎてシャツがぴちぴちだ。 余裕がない。
体のラインがわかるくらい。
ぴったりよりきつい。
服によく使う。
新鮮さを表すときも使う。
ぴったり このサイズはぶかぶかです。25㎝の靴がぴったりだと思います。 大きくも、小さくもなく、多くも、少ない、ちょうどいいさま。
時間、服、量、サイズなどに使う。

✨空間２余裕がある✨

がらがら

すかすか

ぶかぶか

だぶだぶ

単語 例文 説明
がらがら このサッカースタジアムはいつもガラガラだ。 乗り物、建物に人がほとんどいない様子。
がらがらの音。
のどがガラガラ。
すかすか 私のかばんは、いつもすかすかで筆箱しか入っていない。 結構空間が残っていて、まだ人／物が入る。
本棚、箱など。
ぶかぶか この子供はぶかぶかの服を着ている。 身に着けている服や靴が大きすぎるてあまっているようす。
ちょっとだらしない。
だぶだぶ 去年までは、ぴったりの服が流行っていたけど、最近はだぶだぶの服が人気だ。 靴には使えない。
服が長すぎる。

✨からだ１太っている✨

ぶよぶよ

ぱんぱん

ぷにぷに

ぶくぶく

ぽっちゃり

単語 例文 説明
ぶよぶよ もうすぐ夏だから、ぷよぷよしているお腹をなくさないと。 太り過ぎて肉がたるんでいる様子。
ぱんぱん おなかがパンパンになるあで食べ過ぎるのはあまり良くない。腹八分目がちょうど良い。 お腹が破裂しそうなくらい。
ぷにぷに 赤ちゃんのほっぺたはぷにぷにしていて、触り心地がいい。 柔らかく、弾力があるさま。
ぶくぶく お正月、食べ過ぎてぶくぶく太ってしまった。 短い間に太っていく様子。
歯を磨いた後ぶくぶく。
ぽっちゃり やせている女の人より、ぽっちゃりしている方が好きなんだよね。 少し太っている人のことをかわいく表す言葉。

単語 例文 説明
がりがり あの人はがりがりだけど、たくさん食べる。 骨が見えるぐらい瘦せている様子。
硬いものをかみ砕いた時にでる音。
ひょろひょろ ひょろひょろした男は筋トレをしよう。 体が細長くて、力が弱い感じ。
もやしのような食べ物にも使う。
げっそり 一週間あまり食べなかったからげっそりした。 疲れて顔が瘦せた感じ。
顔に使う表現。
良くない意味。
すらっと この人はモデルみたいにすらりとしている。うらやましい。 背が高くて細い。
褒め言葉。
すらっと／すらりと。
ほっそり 二重あごをどうにかしたいです。顔をほっそりさせる方法を教えてください。 ただスリムで細いという意味。
顔、足などによく使う。

へとへと 立って10時間働いた。もうへとへとで歩けないほどだ。 かなり疲れた状態。
基本的には人に対して使う。
くたくた 毎日残業でくたくただ。 かなり疲れ状態＝へとへと
くたくた＜くたばる
ふらふら 昨日から何も食べてなくて、もうふらふらだよ。 不安定に揺れるさま。
しっかり立っていられない。
くらくら 最近くらくらして、吐き気がします。まるで目が回っている感じです。 目まいがするさま。
頭がくらくらする。
がくがく 富士山の頂上に登った。気持ちいいが、膝ががくがくする。 恐怖や疲労などで体の一部分が強く小刻み震えるさま。

力のないさま。

ドアにも使える。
