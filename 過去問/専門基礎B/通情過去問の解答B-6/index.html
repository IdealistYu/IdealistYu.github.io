​ (b)

In Kruskal's algorithm, a common method for determining whether adding a new edge will create a cycle is to use the Union-Find data structure. Specifically, we maintain a Union-Find set where each set represents a connected component. Initially, each vertex is in a separate set. For each edge, we use the Find and Union operations of the Union-Find data structure to determine whether the two vertices it connects belong to the same connected component. If they do, adding the edge would create a cycle and the edge should be discarded. Otherwise, the edge is added to the minimum spanning tree and the two vertices are merged into the same connected component.

​ (a)

To prove that the graph \((V,T \cup\{e\})\) has exactly one cycle, we need to show that it satisfies the two conditions for a graph to be a cycle:

  1. The graph has no more than one cycle.
  2. The graph has at least one cycle.

First, let's prove that \((V,T \cup\{e\})\) has no more than one cycle. Since \(T\) is a spanning tree of \(G\), it is acyclic, which means it has no cycles. When we add the edge \(e\) to \(T\), we create exactly one cycle, because there is exactly one path from one endpoint of \(e\) to the other in \(T\). Adding \(e\) to \(T\) creates a unique cycle, which means \((V,T \cup\{e\})\) has no more than one cycle.

Now, let's prove that \((V,T \cup\{e\})\) has at least one cycle. Since \(T\) is a spanning tree of \(G\), it contains all vertices in \(V\) and has \(|V| - 1\) edges. When we add the edge e to \(T\), we get a connected graph with \(|V|\) vertices and \(|V|\) edges. Since the graph has one more edge than a tree, it must contain a cycle. Therefore, \((V,T \cup\{e\})\) has at least one cycle.

Combining these two proofs, we can conclude that \((V,T \cup\{e\})\) has exactly one cycle.

​ (b)

The two propositions (i) and (ii) are equivalent.

To prove that (i) implies (ii), suppose that \(T\) is a spanning tree of \(G\) with minimum weight. Let \(f\) be an edge of \(E\) that is not included in \(T\). By adding \(f\) to \(T\), we obtain a unique cycle \(C = F(T, f)\). Assuming there exists an edge \(e'\) in cycle \(C\) such that \(w(e') > w(f)\), then there exists a spanning tree \(T'\) (containing edge \(f\) but not \(e'\)) such that \(w(T') < w(T)\), which contradicts the assumption that \(T\) is a minimum weight spanning tree. Therefore,we mush have \(w(e') ≤ w(f)\) for every edge \(e'\) in \(C = F(T', f)\), which proves (ii).

To prove that (ii) implies (i), suppose that (ii) holds. suppose that \(T\) is a spanning tree of \(G\). Let \(f\) be an edge of \(E\) that is not included in \(T\). By adding \(f\) to \(T\), we obtain a unique cycle \(C = F(T, f)\). For every cycle \(C\), we have \(w(e') ≤ w(f)\). Assuming the spanning tree \(T'\) contains \(f\), then \(w(T') ≥ w(T)\). Therefore, \(T\) must be a minimum weight spanning tree, which proves (i).

Hence, we have shown that (i) and (ii) are equivalent.

2019解答

（１） \[ \begin{align} 完全ではないマッチング&:\{\{1,2\},\{3,4\}\} \\ 完全マッチング&:\{\{1,2\},\{3,4\},\{7,8\},\{5,9\},\{15,16\},\{17,18\},\{14,6\},\{10,11\},\{12,13\}\} \end{align} \] （２） \[ \begin{align} X(G_1,S_1)=2 \end{align} \] （３） \[ \begin{align} |E_{G_1}(S_1,V(G_1)\backslash S_1)|=8 \end{align} \] （４）

Consider a graph \(G\), with a perfect matching. Let \(S\) be an arbitrary subset of \(V\). Delete \(S\). Let \(C\) be an arbitrary odd component in \(G − S\). Since \(G\) had a perfect matching, at least one vertex in \(C\) must be matched to a vertex in \(S\). Hence, each odd component has at least one vertex matched with a vertex in \(S\). Since each vertex in \(S\) can be in this relation with at most one connected component (because of it being matched at most once in a perfect matching), \(X(G − S) ≤ |S|\).

グラフ \(G\) に完全マッチングがあるとする。\(S\) を \(V\) の任意の部分集合として、グラフから \(S\) を除く。\(C\) を 誘導部分グラフ \(G − S\) の任意の奇頂点連結成分とする。\(G\) は完全マッチングを持つから、\(C\) にはそのマッチングにおいて \(S\) の要素と接続されていた頂点が少なくとも\(1\)つは存在する。

この対応付けによって \(G − S\) の奇頂点連結成分全体から \(S\) への写像が定まるが、これはマッチングの完全性より単射である。よって \(X(G − S) ≤ |S|\)。

（５）

Assuming that \(E\) is the set of edges in \(C\), and \(v\) is an arbitrary vertex in \(C\), then \(\sum\limits_{v\in C} \mathrm {deg}(v)=2|E|\). Consider the set \(V\) formed by adding the edges between \(C\) and \(S\) to \(C\), the total degree of \(V\) is \(2|E|+E_G(S,C)\). Also, since \(G\) is a 3-regular graph, the total degree of \(V\) is \(3|C|\). Therefore, \(3|C|=2|E|+|E_G(S,C)|\). If \(|C|\) is odd, then \(3|C|\) is odd. Since \(2|E|\) is even, \(|E_G(S,C)|\) is odd.
